# Rodando o servidor

Comandos importantes:

```sh
# Ter o node instalado
# Instalar pacotes
npm i

# Configurar o .env a partir do .env.local-Example

# Criar banco de dados
npm run migrate

# (OPCIONAL - RODAR A SEED)
npm run seed

# Fazer a build do next
npm run build
npm start #(sรณ pra testar na maquina local se a build ta funcionando)

#O deploy do projeto fica para mais pra frente!
```
